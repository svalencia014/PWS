Create Flight Plan

  • IFR: CALLSIGN + SQWK +SCRATCHPAD ACTYPE CRUISEALT DEP*ARR

    Ex: DAL4205 + 2000 +ARN A20N 310 JAX*ATL

  • VFR: CALLSIGN / 1200 +SCRATCHPAD ACTYPE CRUISEALT DEP*ARR

    Ex: N700MS / 1200 +PTN C172 015 JAX*JAX

Set Altitude Filters

  • All Targets: F7 + F LOWALTHIGHALT LOWALTHIGHALT

    Ex: F7 + F 000160 000160

  • Associated Only: F7 + FC LOWALTHIGHALT

    Ex: F7 + FC 000160

GI Text

  • Set Atis Code: F7 + S${A-Z} optionalText

    Ex: F7 + SJ 08/14

    Ex: F7 + SD 7L/R

  • Update Atis Code: F7 S${A-Z}

    Ex: F7 + SM
